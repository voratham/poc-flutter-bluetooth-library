## How to run test & build
### Prerequisites
- Install Flutter SDK (i using Flutter 3.29.3)
- Install Android Studio


```sh
# cd test_flutter_reactive_ble or test_flutter_blue_plus

flutter pub get
#  open android simulator or connect to a real device
flutter run

flutter build apk
```
